This is the dashboard tab.
It will display stats about the server.
This includes the player count, RAM usage, CPU usage and clock speed, etc.

This is the players tab.
It will have settings to manage players, including the whitelist and blacklist.
Sliders should be added to op and deop players as well as select their op level.
This will edit ops.json

This is the settings tab.
It should provide fields to change the values in server.properties

This is the worlds tab.
Players should be able to upload a zip file of their custom worlds.
Already uploaded worlds should be displayed in a list and should be selectable.
A server restart will need to take place.

This is the terminal tab.
It will provide access to the server terminal and log.
